# Portfolio Website Design System

## Design Philosophy

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) and pure white (#ffffff) for high contrast
- **Accent**: Warm amber (#f59e0b) for highlights and interactive elements
- **Secondary**: Soft gray (#6b7280) for supporting text and subtle elements
- **Background**: Gradient from deep charcoal to black (#0f0f0f to #000000)

### Typography
- **Display Font**: "Playfair Display" - elegant serif for headings and hero text
- **Body Font**: "Inter" - clean sans-serif for readability and modern feel
- **Code Font**: "JetBrains Mono" - for technical elements and code snippets

### Visual Language
- **Minimalist Aesthetic**: Clean lines, generous whitespace, focused content
- **Professional Yet Creative**: Balancing technical expertise with artistic flair
- **High Contrast**: Ensuring accessibility with 4.5:1 contrast ratios
- **Subtle Animations**: Enhancing user experience without overwhelming

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **Typed.js**: Typewriter effect for dynamic text in hero section
- **Splitting.js**: Advanced text animations and stagger effects
- **ECharts.js**: Data visualization for skills and project metrics
- **p5.js**: Creative coding for background particle effects

### Animation Effects
- **Hero Typewriter**: Dynamic role display with cursor animation
- **Scroll Reveals**: Content fades in as user scrolls down the page
- **Hover Transformations**: 3D tilt effects on cards and buttons
- **Particle Background**: Subtle floating particles for visual depth
- **Skill Progress Bars**: Animated filling effects on scroll

### Styling Approach
- **Glassmorphism**: Semi-transparent elements with backdrop blur
- **Gradient Overlays**: Subtle gradients for depth and visual interest
- **Box Shadows**: Layered shadows for elevated card effects
- **Border Radius**: Consistent 8px radius for modern rounded corners
- **Grid Layouts**: CSS Grid for responsive, structured content

### Header Effect
- **Animated Navigation**: Smooth underline transitions on hover
- **Sticky Header**: Transparent background with blur effect
- **Logo Animation**: Subtle scale and glow on interaction
- **Mobile Menu**: Slide-out drawer with staggered item animations

### Interactive Elements
- **Button Hover**: Color morphing with shadow expansion
- **Card Interactions**: Lift and tilt effects on hover
- **Form Focus**: Input field highlighting with smooth transitions
- **Image Galleries**: Ken Burns effect on portfolio images
